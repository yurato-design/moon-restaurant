# MoonRestaurant｜月明かりをテーマにしたレストランサイト（デザイン・コーディング）

月の出のような柔らかなオレンジを基調に、  
月明かりがそっと照らすような温かい雰囲気をイメージして制作したデモサイトです。  
トップページでは静かな夜の空気感を感じられるビジュアルを配置し、  
CONCEPT / MENU / INFORMATION へ自然にアクセスできるよう視線の流れを設計しました。

スマートフォンでの閲覧を前提に、情報量を整理しながら読みやすさを重視したレイアウトを採用。  
ハンバーガーメニューや写真ギャラリーなど、夜の落ち着いた世界観を損なわないシンプルな UI を実装しています。

HTML / CSS を中心に制作し、レスポンシブ対応、画像最適化、OGP 設定など、  
実務で求められる基本要素を押さえながら、デザイン〜コーディングまで一貫して制作したデモサイトです。

## 📸 サイト構成

- **TOP（index.html）**  
  月明かりをイメージしたメインビジュアルと 3 つの導線

- **CONCEPT（concept.html）**  
  写真とテキストで世界観を伝えるコンセプトページ

- **MENU（menu.html）**  
  サイド・メイン・アラカルトの写真メニュー一覧

- **INFO（info.html）**  
  写真ギャラリー、店舗情報、アクセスマップ
